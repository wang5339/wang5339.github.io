使用Git将本地文件上传到github的步骤

一，先建立git仓库

复制上面的网址备用

打开git bash，如何一直使用cd到本地项目根目录，再执行git init

** (git init//把这个目录变成git可以管理的仓库)**

二，将项目的文件添加到仓库中

执行**git add .**
**(git add README.md//文件添加到仓库)**
**(git add .//不但可以跟单一文件，还可以跟通配符，更可以根目录。一个点就把当前目录下所以未追逐的文件全部add了）**

三，将add文件commit到仓库

执行**git commit -m**“这里写注释语句”//把文件提交到仓库

四，将本地仓库关联到github上

执行**git remote add origin** 这里输入刚才复制的仓库地址

五，上传前，要先pull一下，执行以下命令

**git push origin master**

六，上传代码到github远程仓库

执行**git push -u origin master**//把本地库的使用内容推送到远程库上

七，正常操作可能会出现的问题

中间可能会让你输入Username和Password
如果出现下面问题按照提示输入邮箱跟用户名
